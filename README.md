# Wichtige Formeln für Elektroinstallateur-Lehrlinge

Diese Website enthält die wichtigsten Formeln für Elektroinstallateur-Lehrlinge. Jede Formel ist auf einer Karte dargestellt. Klicke auf eine Karte, um sie umzudrehen und die Erklärung sowie mögliche Umformungen zu sehen.

## Verwendung

Öffne `index.html` in einem Webbrowser, um die Website zu starten. Die Karten sind interaktiv und drehen sich beim Klick.

## Technologien

- HTML
- CSS
- JavaScript

## Formeln enthalten

- Ohm'sches Gesetz
- Leistung
- Widerstand in Reihe und parallel
- Kapazität in Reihe und parallel
- Induktivität in Reihe und parallel
- Effektivwert (Wechselstrom)
- Leistungsfaktor

## Anpassungen

Du kannst weitere Formeln hinzufügen, indem du die `index.html` bearbeitest und neue `.flip-card` Elemente hinzufügst.
