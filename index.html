⚡ Ohm'sches Gesetz

U = R × I

Ohm'sches Gesetz

Die Spannung U ist gleich dem Produkt aus Widerstand R und Stromstärke I.

Umformungen:

  • I = U / R
  • R = U / I

💡 Elektrische Leistung

P = U × I

Elektrische Leistung

Die Leistung ist die Energie pro Zeit.

Umformungen:

  • P = R × I²
  • P = U² / R

🔌 Elektrische Energie

W = P × t

Elektrische Energie (Arbeit)

Energie ist Leistung mal Zeit.

Umformungen:

  • W = U × I × t

🧲 Widerstand (Material)

R = ρ × (l / A)

Widerstand (Material)

Der Widerstand hängt vom spezifischen Widerstand ρ, der Länge l und dem Querschnitt A ab.

  • ρ = spezifischer Widerstand
  • l = Länge des Leiters
  • A = Querschnittsfläche

🔄 Reihenschaltung

Rges = R1 + R2 + ...

Reihenschaltung

Bei einer Reihenschaltung addieren sich die Widerstände und Spannungen.

  • Rges = R1 + R2 + R3 + ...
  • Uges = U1 + U2 + U3 + ...
  • Iges = I1 = I2 = I3 (konstant)

🔀 Parallelschaltung

1/Rges = 1/R1 + 1/R2 + ...

Parallelschaltung

Bei einer Parallelschaltung addieren sich die Leitwerte (1/R) und die Ströme.

  • 1/Rges = 1/R1 + 1/R2 + 1/R3 + ...
  • Uges = U1 = U2 = U3 (konstant)
  • Iges = I1 + I2 + I3 + ...

⚡ Ladung

Q = I × t

Ladung

Die Ladung ist Stromstärke mal Zeit.

  • Q = Ladung (Coulomb)
  • I = Stromstärke (Ampere)
  • t = Zeit (Sekunden)

🔌 Spannung (Definition)

U = W / Q

Spannung (Definition)

Spannung ist Energie pro Ladung.

  • U = Spannung (Volt)
  • W = Energie (Joule)
  • Q = Ladung (Coulomb)

🧠 Wirkungsgrad

η = Pab / Pzu

Wirkungsgrad

Der Wirkungsgrad beschreibt, wie viel der zugeführten Leistung in nutzbare Leistung umgewandelt wird.

  • η = Wirkungsgrad (0-1 oder %)
  • Pab = abgegebene Leistung
  • Pzu = zugeführte Leistung
