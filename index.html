⚡ Ohm'sches Gesetz

U = I × R

Ohm'sches Gesetz

Die Spannung U ist gleich dem Produkt aus Stromstärke I und Widerstand R.

Umformungen:

  • I = U / R
  • R = U / I

💡 Leistung

P = U × I

Leistung

Die Leistung P ist die Energie pro Zeit. Sie kann auf verschiedene Weisen berechnet werden.

Umformungen:

  • P = I² × R
  • P = U² / R
  • U = P / I
  • I = P / U
  • R = P / I²
  • R = U² / P

🔌 Widerstand in Reihe

R_ges = R1 + R2 + ...

Widerstand in Reihe

Bei Reihenschaltung addieren sich die Widerstände.

Keine Umformungen nötig, da es die Grundform ist.

🔌 Widerstand parallel

1/R_ges = 1/R1 + 1/R2 + ...

Widerstand parallel

Bei Parallelschaltung addieren sich die Leitwerte (1/R).

Umformung: R_ges = 1 / (1/R1 + 1/R2 + ...)

🔋 Kapazität parallel

C_ges = C1 + C2 + ...

Kapazität parallel

Bei Parallelschaltung addieren sich die Kapazitäten.

Keine Umformungen nötig.

🔋 Kapazität in Reihe

1/C_ges = 1/C1 + 1/C2 + ...

Kapazität in Reihe

Bei Reihenschaltung addieren sich die Kehrwerte der Kapazitäten.

Umformung: C_ges = 1 / (1/C1 + 1/C2 + ...)

🌀 Induktivität in Reihe

Induktivität in Reihe

Keine Umformungen nötig.

🌀 Induktivität parallel

1/L_ges = 1/L1 + 1/L2 + ...

Induktivität parallel

Bei Parallelschaltung addieren sich die Kehrwerte der Induktivitäten.

Umformung: L_ges = 1 / (1/L1 + 1/L2 + ...)

📊 Effektivwert (Wechselstrom)

U_eff = U_max / √2

Effektivwert (Wechselstrom)

Der Effektivwert ist der quadratische Mittelwert der Spannung.

Umformungen:

  • U_max = U_eff × √2
  • Gleiches gilt für Strom: I_eff = I_max / √2

⚙️ Leistungsfaktor

cos φ = P / S

Leistungsfaktor

Der Leistungsfaktor gibt das Verhältnis von Wirkleistung P zu Scheinleistung S an.

Umformungen:

  • P = S × cos φ
  • S = P / cos φ
