⚡ Ohm'sches Gesetz

U = R × I

Ohm'sches Gesetz

Die Spannung U ist gleich dem Produkt aus Widerstand R und Stromstärke I.

Umformungen:

  • I = U / R
  • R = U / I

💡 Elektrische Leistung

P = U × I

Elektrische Leistung

Die Leistung ist die Energie pro Zeit.

Umformungen:

  • P = U × I
  • P = R × I²
  • P = U² / R
  • U = P / I
  • I = P / U
  • R = U² / P
  • R = P / I²

🔌 Elektrische Energie

W = P × t

Elektrische Energie (Arbeit)

Energie ist Leistung mal Zeit.

Umformungen:

  • W = P × t
  • W = U × I × t
  • P = W / t
  • t = W / P

🧲 Widerstand (Material)

R = ρ × (l / A)

Widerstand (Material)

Der Widerstand hängt vom spezifischen Widerstand ρ, der Länge l und dem Querschnitt A ab.

Umformungen:

  • R = ρ × (l / A)
  • ρ = R × A / l
  • l = R × A / ρ
  • A = ρ × l / R

🔄 Reihenschaltung

Rges = R1 + R2 + ...

Reihenschaltung

Bei einer Reihenschaltung addieren sich die Widerstände und Spannungen.

  • Rges = R1 + R2 + R3 + ...
  • Uges = U1 + U2 + U3 + ...
  • Iges = I1 = I2 = I3 (konstant)

🔀 Parallelschaltung

1/Rges = 1/R1 + 1/R2 + ...

Parallelschaltung

Bei einer Parallelschaltung addieren sich die Leitwerte (1/R) und die Ströme.

  • 1/Rges = 1/R1 + 1/R2 + 1/R3 + ...
  • Uges = U1 = U2 = U3 (konstant)
  • Iges = I1 + I2 + I3 + ...

⚡ Ladung

Q = I × t

Ladung

Die Ladung ist Stromstärke mal Zeit.

Umformungen:

  • Q = I × t
  • I = Q / t
  • t = Q / I

🔌 Spannung (Definition)

U = W / Q

Spannung (Definition)

Spannung ist Energie pro Ladung.

Umformungen:

  • U = W / Q
  • W = U × Q
  • Q = W / U

🧠 Wirkungsgrad

η = Pab / Pzu

Wirkungsgrad

Der Wirkungsgrad beschreibt, wie viel der zugeführten Leistung in nutzbare Leistung umgewandelt wird.

Umformungen:

  • η = Pab / Pzu
  • Pab = η × Pzu
  • Pzu = Pab / η

📋 Stromlaufschema

Einpolig & Mehrpolig

Stromlaufschema

Einpoliges Schema: Zeigt nur eine Phase (L) + Neutralleiter (N)

Mehrpoliges Schema: Zeigt alle 3 Phasen (L1, L2, L3) + N

  • Einpolig: vereinfachte Darstellung
  • Mehrpolig: vollständige Darstellung
  • Für Drehstrom (3~): Mehrpolig notwendig

🔌 Ein/Aus-Schaltung

Sch1

Ein/Aus-Schaltung (Sch1)

Die einfachste Schaltung: ein Schalter steuert ein Gerät.

  • Sch1: Ein Schalter trennt/verbindet Stromkreis
  • Schalter in Serie mit dem Verbraucher
  • Anwendung: Lampe, Motor, einfaches Gerät
  • Symbol: Einfacher Schalter ─o─

⛓️ Serienschaltung

Reihenschaltung

Serienschaltung (Reihenschaltung)

Mehrere Verbraucher hintereinander geschaltet.

  • Widerstände addieren: Rges = R1 + R2 + R3
  • Strom ist überall gleich: I = I1 = I2
  • Spannungen addieren: U = U1 + U2
  • Nachteil: Wenn eine Lampe ausfällt, alle aus

🔄 Wechselschaltung

Sch2

Wechselschaltung (Sch2)

Zwei Schalter an verschiedenen Orten steuern das gleiche Gerät.

  • Sch2: 2 Umschalter an 2 verschiedenen Orten
  • Von beiden Stellen EIN und AUS möglich
  • Anwendung: Treppenhaus, Flur (2 Lichtschalter)
  • Verdrahtung: Cross-Connection zwischen den Schaltern

✕ Kreuzschaltung

Sch3

Kreuzschaltung (Sch3)

Drei oder mehr Schalter an verschiedenen Orten steuern das gleiche Gerät.

  • Sch3: Mindestens 2 Wechselschalter + Kreuzschalter(n)
  • Von allen Stellen EIN und AUS möglich
  • Anwendung: Lange Flure, Galerien, 3+ Etagen
  • Der Kreuzschalter invertiert das Signal

🔗 Reihen- & Parallelschaltung

Sch5 & Sch6

Reihen- & Parallelschaltung

Sch5 - Reihenschaltung: Mehrere Verbraucher in Reihe

Sch6 - Parallelschaltung: Mehrere Verbraucher parallel

  • Sch5 (Reihe): Rges = R1 + R2 + R3
  • Sch6 (Parallel): 1/Rges = 1/R1 + 1/R2
  • Sch5: Wenn eine Lampe ausfällt, alle aus
  • Sch6: Unabhängige Verbraucher, praktisch in Häusern

📋 Schemaarten - Übersicht

Die folgenden Schemaarten sind standardisierte Schaltschemen in der Elektrotechnik:

Sch1: Ein/Aus-Schaltung

Ein einzelner Schalter steuert einen Verbraucher (z.B. Lampe).

  • Einfachste Form
  • 1 Schalter + 1 Verbraucher
  • Anwendung: Einfache Deckenlampe

Sch2: Wechselschaltung

Zwei Schalter an verschiedenen Orten steuern einen Verbraucher unabhängig voneinander.

  • 2 Umschalter (Wechselschalter)
  • Von beiden Orten EIN/AUS möglich
  • Anwendung: Treppenhaus, Langer Flur

Sch3: Kreuzschaltung

Drei oder mehr Schalter an verschiedenen Orten steuern einen Verbraucher unabhängig.

  • 2 Wechselschalter + mindestens 1 Kreuzschalter
  • Von allen Orten EIN/AUS möglich
  • Anwendung: Lange Flure, mehrere Etagen

Sch5: Reihenschaltung

Mehrere Verbraucher hintereinander in einer Leitung verbunden.

  • Widerstände addieren sich: Rges = R1 + R2
  • Strom überall gleich
  • Problem: Ausfallrisiko (eine Lampe = alle aus)

Sch6: Parallelschaltung

Mehrere Verbraucher parallel zueinander verbunden.

  • Leitwerte addieren: 1/Rges = 1/R1 + 1/R2
  • Spannung überall gleich
  • Vorteil: Unabhängiger Betrieb (Standard in Häusern)
